# UpcycleConnect — PA2526

Application web de mise en relation entre particuliers et prestataires
pour la réparation, la transformation et le recyclage d'objets du quotidien.

Projet annuel ESGI 2A — 2025-2026.

## Team Members

- Youssouf Banhoro DIARRA
- Sofiane YALA
- Jaydee PHANOUKOUN

## Stack technique

- Backend API : Go
- Frontend : PHP MVC custom, Tailwind CSS, DaisyUI
- Base de données : MySQL

## Repository Structure

- /api : Backend API Go
- /frontend : Frontend PHP MVC
- /schema : Schémas BDD et réseau
- /docs : Documents fonctionnels et cahier des charges
- /devis : Estimations financières
- /trello : Organisation projet
